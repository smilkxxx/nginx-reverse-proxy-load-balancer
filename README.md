# 🚀 Reverse Proxy y Balanceo de Carga con Nginx

## 📖 Descripción

Este proyecto consiste en la implementación de una infraestructura web utilizando **Nginx** como **Reverse Proxy** y **Balanceador de Carga**, distribuyendo las solicitudes entre dos servidores **Apache**.

El laboratorio fue desarrollado sobre Linux con el objetivo de demostrar el funcionamiento de una infraestructura web distribuida, fortaleciendo conocimientos en administración de servidores, balanceo de carga y servicios web.

---

## 🛠️ Tecnologías utilizadas

- Linux
- Nginx
- Apache2
- WordPress
- Microsoft SQL Server
- VirtualBox

---

## ✅ Funcionalidades

- Configuración de Nginx como Reverse Proxy.
- Implementación de Balanceo de Carga entre dos servidores Apache.
- Despliegue de WordPress.
- Integración con Microsoft SQL Server.
- Administración de servicios Linux.
- Validación del balanceo mediante pruebas de acceso.

---

## 🎯 Objetivos

- Implementar un Reverse Proxy utilizando Nginx.
- Configurar un Balanceador de Carga.
- Comprender el funcionamiento de una arquitectura distribuida.
- Administrar múltiples servidores Linux.
- Fortalecer conocimientos en infraestructura web y alta disponibilidad.
